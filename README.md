Проект Kanban Desk.

Технологии:
React, Create React App.
Использованные хуки React: useState, useEffect, useRef, usePortal.

Функционал:
Доска состоит из 4 статусов: "Backlog", "Ready", "In Progress", "Finished".
Создание новых задач доступно в статусе "Backlog".
Добавление задач в остальные статусы происходит через выбор задачи из списка предыдущего статуса.
При клике на задачу доступно подробное описание и возможность редактирования описания задачи (двойной клик).
При необходимости есть возможность удалить задачу с доски.
Задачи сохраняются в локальном хранилище браузера.
